## Быстрый старт

```bash
# 1. Скачайте скрипт
curl -fsSL -o setup.sh https://raw.githubusercontent.com/MarkReeR/sh/main/setup.sh

# 2. Сделайте исполняемым
chmod +x setup.sh

# 3. Запустите (требуется root)
sudo ./setup.sh
